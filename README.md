# Alternate Text Manager

A production-ready React + TypeScript application for managing AI-generated alternate text for images.

---

## Setup Instructions

### Prerequisites
- Node.js >= 16.x
- npm >= 8.x

### Installation

```bash
git clone <your-repo-url>
cd alt-text-manager
npm install
npm start
```

## Key Features

- Filter by Image Status, Type, and Complexity (cumulative)
- Master + individual checkbox selection
- Lazy-loaded image thumbnails
- Paginated data grid (Previous/Next + page numbers)
- Loading states and empty state handling
